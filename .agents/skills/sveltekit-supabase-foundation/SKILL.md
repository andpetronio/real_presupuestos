# sveltekit-supabase-foundation

## Purpose

Set up and maintain the technical foundation of the SvelteKit application with Supabase as the primary backend platform.

## Use this skill when
- bootstrapping the app
- configuring Supabase
- configuring internal auth
- defining project structure
- defining environment variables
- setting up protected internal routes

## Core rules
- Use SvelteKit with TypeScript
- Use Supabase for database and internal authentication
- Keep business logic outside components
- Use authenticated internal routes and public token routes separately
- Keep environment-driven configuration

## Deliverables
- base project structure
- Supabase SSR integration
- internal route protection
- public unauthenticated route support
- documented environment conventions

## Do not use this skill for
- budgeting formulas
- multi-dog workflow logic
- public accept/reject domain rules
- WhatsApp provider implementation
