# sveltekit-admin-crud

## Purpose

Implement standard internal CRUD flows for master data inside the authenticated SvelteKit application.

## Use this skill when
- creating tutors screens
- creating dogs screens
- creating raw materials screens
- creating recipes screens
- creating settings screens

## Core rules
- Keep CRUD simple and readable
- Use server-side modules for persistence
- Avoid heavy business logic in page components
- Build forms for operator clarity
- Prioritize responsiveness and low latency

## Deliverables
- list pages
- create/edit forms
- view/detail pages where useful
- validation and user-friendly error handling

## Do not use this skill for
- complex budget workflow
- historical snapshots
- WhatsApp sending
- public token response pages
